# 部署设置

本项目默认保留 Cloudflare/OpenNext 部署链路，但不保留任何旧项目域名、worker 名、zone 信息或 secret。

## 必须替换

- Cloudflare account / worker / route
- 正式域名
- Turnstile site key 和 secret
- 邮件服务配置
- 表单接收配置

## 本地文件

这些文件只在本地创建，不入库：

- `.env.local`
- `.dev.vars`
- `.mcp.json`

## 验证命令

```bash
pnpm build
pnpm build:cf
pnpm release:verify
```
